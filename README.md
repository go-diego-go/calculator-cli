# Fraction calculator

This is a command line application that can perform the following operations on fractions:

* Add
* Substract
* Multiply
* Divide

It supports improper fractions, mixed numbers and whole numbers.

## Example usage

```
? 1/2 * 3_3/4

= 1_7/8
```
 
```
? 2_3/8 + 9/8

= 3_1/2
```

## Installation

* `brew install pipenv` 
* `make install`

## Execution

* `calculator`

## Development

* Run test scripts with `make qa`
